# file-search-api
generate answer by file-search-api

## 환경 정보
```
pip install pandas
pip install tqdm
pip install google-genai
pip install openpyxl
pip install tabulate
pip install python-dotenv

pip freeze > requirements.txt
```

## 환경 세팅
```bash
cd file_search_api_space
python3 -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt
```
### 파일 실행

```bash
python3 -m file_search_api.py
```
